This repository contains two files.

1.  run_analysis.R 
    The script run_analysis.R downloads an archived file containing data obtained
    from a wearable computers.  It combines test and train datasets into one merged
    dataset, averages for each subject, activity, and measurement.  It then writes the
    valuess to a file "getting_data_output.txt".

2.  CODEBOOK.md
    This codebook contains a description of the analysis run by run_analysis.R as well
    as descriptions of the variables contained in getting_data_output.txt.
